# Mos 4.0.0 (20260220.1) - TengJoe Codex Fix

这个版本基于上游 `Mos 4.0.0 (20260220.1)`，不增加新功能，重点是几项针对性的稳定性和兼容性修复。

## 亮点

- 通过在跨线程前复制滚动事件，缓解 `CVDisplayLink` 相关滚动闪退路径
- 通过收窄 Launchpad 检测逻辑，恢复 Dock 文件夹和叠放视图中的平滑与反转滚动
- 通过识别 `com.apple.ScreenContinuity`，改善 iPhone 镜像中的反转滚动处理
- 通过把 `com.netease.uuremote` 识别为远控应用，避免 UU 远程桌面双重平滑

## 包身份

- fork：`TengJoe/Mos-fix`
