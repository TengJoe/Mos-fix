# Mos 4.0.0 (20260220.1) - TengJoe Codex Fix

This release is based on upstream `Mos 4.0.0 (20260220.1)` and keeps the original feature set while adding a small set of targeted fixes for stability and compatibility.

## Highlights

- mitigates the `CVDisplayLink` scrolling crash path by copying the event before it crosses thread boundaries
- restores smooth and reverse scrolling in Dock folders and stack views by narrowing Launchpad detection
- improves reverse-scroll handling in iPhone Mirroring by recognizing `com.apple.ScreenContinuity`
- avoids double smoothing in UU Remote Desktop by treating `com.netease.uuremote` as a remote-control app
